# 📚 Introdução a APIs

Este repositório tem como objetivo apresentar os conceitos básicos de **APIs (Application Programming Interfaces)**, com exemplos práticos e simples para quem está começando no desenvolvimento backend ou consumo de serviços web.

---

## 🚀 O que é uma API?

Uma API é uma interface que permite a comunicação entre sistemas diferentes. Ela define como os dados podem ser enviados e recebidos entre aplicações.

Exemplo comum:
- Um aplicativo de clima que busca dados de um servidor externo
- Um sistema de login usando autenticação via API

---

## 🧠 Conceitos abordados

- O que é uma API REST
- Métodos HTTP:
  - `GET` (buscar dados)
  - `POST` (criar dados)
  - `PUT` (atualizar dados)
  - `DELETE` (remover dados)
- Status codes (200, 201, 400, 404, 500)
- JSON como formato de dados
- Consumo de APIs com requisições HTTP
